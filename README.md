# Go Dynamic Web Server

A simple Go web server that displays dynamic content (current server time).

## About This Project

This project was created as part of the Moringa AI Capstone Project to demonstrate learning a new technology (GoLang) using Generative AI prompts.

## Features

- ✅ Displays current server time dynamically
- ✅ Built with Go's standard library (no external dependencies)
- ✅ Simple and beginner-friendly code
- ✅ Cross-platform compatible

## Quick Start

### Prerequisites
- Go 1.21 or higher installed
- Web browser

### Installation

1. Clone this repository:
```bash
git clone https://github.com/yourusername/go-dynamic-webserver.git
cd go-dynamic-webserver
```

2. Run the server:
```bash
go run main.go
```

3. Open your browser and visit:
```
http://localhost:8080
```
